Python 3.11新特性

Python 3.11引入了一些实用的新特性，最值得关注的有：

1）性能提升：这是Python 3.11最大的亮点。通过优化解释器和运行时，Python 3.11比3.10快10-60%。如果你的项目对性能有要求，升级到3.11会有明显改善。

2）更好的错误信息：错误信息现在更加详细和准确。当代码出错时，Python会明确指出错误发生的位置，甚至给出修改建议。

3）异常组和except*：新增了ExceptionGroup和except*语法，可以同时处理多个异常，这在异步编程中特别有用。

4）自包含的解释器：Python 3.11可以构建自包含的解释器，方便部署和分发。
