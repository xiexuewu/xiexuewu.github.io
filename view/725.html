PuTTY基础使用

安装完成后即可开始使用 putty中文版进行远程连接。

打开安装好的 PuTTY ，对照下图，按照如下步骤进行连接：

  • 指定端口，默认是 22端口。(这个端口要和服务器上的配置一致，默认就是 22 端口)

PuTTY常见使用问题

1. putty下载安装后连接失败，提示 "Network error: Connection refused"：这通常表示目标服务器未运行 SSH 服务，或者防火墙规则阻止了连接。需要检查服务器端的 SSH 服务状态，确认防火墙是否开放了相应端口。

2. putty中文版登录时显示 "Access denied" 错误：可能原因包括用户名或密码错误、服务器配置限制该用户登录、或密钥认证配置不正确。需要仔细核对登录凭据，检查服务器上的用户权限设置和认证方式配置。

3. 中文内容显示为乱码：这是因为字符编码设置不匹配。在 putty中文版的配置界面中，进入 Window -> Translation 设置，将 "Remote character set" 改为 UTF-8，可以解决绝大多数中文显示问题。

4. 连接经常超时或意外断开：可能是网络不稳定或服务器设置了会话超时限制。可以在 Connection 设置中调整保持活动间隔（Seconds between keepalives），建议设置为 30-60 秒，防止因连接空闲而被断开。

5. 无法使用 Ctrl+V 粘贴内容：putty中文版默认使用右键进行粘贴操作。如需启用 Ctrl+V 粘贴功能，可以在 Window -> Selection 设置中勾选 "Ctrl+V pastes clipboard content" 选项。

6. putty下载的版本兼容性问题：如果遇到连接问题，可能是客户端与服务器SSH版本不兼容。建议更新到最新版本的putty中文版，并确保服务器SSH服务也保持更新。
