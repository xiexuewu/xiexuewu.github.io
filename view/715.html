Dopamine 是由开发者 Digimezzo 使用 C# 和 WPF 技术开发的 Windows 平台音乐播放器（最新 3.0 版本已支持跨平台）。与其他播放器追求"功能越多越好"不同，Dopamine 的开发理念非常明确："让组织和聆听音乐尽可能简单和美观"。

开发者曾在官网解释为什么要创造 Dopamine：

"市面上有很多音频播放器，它们中的大多数比 Dopamine 拥有更多的功能和更高的可配置性。但这正是困扰我的问题：我想听音乐，而不是花时间配置我的音乐播放器。所以，Dopamine 有一个简单的目的：点燃它，让它变得简单、清晰和美丽。"

这种"反内卷"的设计理念，让 Dopamine 成为了那些只想安静听歌的用户的完美选择。

Dopamine 具备以下核心功能：
1) 全格式音频支持

  • 基于 FFmpeg 解码，确保音质和稳定性

2) 极简美学设计
  • 采用现代 UWP 风格界面，支持浅色/深色主题
  • 可跟随 Windows 10/11 强调色自动切换配色
  • 四种播放界面模式：普通模式、封面模式、简洁模式、极简模式

3) 智能音乐库管理
  • 自动按艺术家、专辑、风格分类整理
  • 智能识别元数据（ID3 标签），自动下载专辑封面
  • 内置标签编辑器，支持批量修改歌曲信息

4) 专业音频调节
  • 内置均衡器，支持自定义音效
  • 对数音量控制，更自然的音量调节体验
  • 无缝播放（Gapless Playback），专辑曲目间无停顿切换

安装Dopamine

1、Windows 安装版安装步骤

步骤 1：下载安装包。下载 Dopamine-3.0.2.exe 文件，建议保存到桌面或下载文件夹。

步骤 2：运行安装程序，双击下载的 .exe 文件。如果 Windows 弹出"Windows 已保护你的电脑"提示，点击"更多信息" → "仍要运行"（这是正常的，因为软件未进行昂贵的数字签名认证，但完全安全）

步骤 3：选择安装位置。默认路径：C:\Program Files\Dopamine\，建议修改： 将 C 盘改为 D 盘或其他数据盘，例如 D:\Dopamine\，点击"浏览"选择自定义路径，或直接修改路径文本，点击"安装"


步骤 4：完成安装，安装过程通常只需 10-30 秒，点击"完成"。

2、便携版使用方法

如果你下载的是 .zip 压缩包：

步骤 2：运行软件。打开解压后的文件夹，双击 Dopamine.exe 即可启动。

常见问题与解决方案

Q1：为什么有些歌曲没有正确归类到同一张专辑？

原因： 这是元数据标签问题。Dopamine 根据"专辑名称"和"专辑艺术家"两个标签来组织专辑。如果合辑（Various Artists）中的每首歌都标记了不同的"专辑艺术家"，Dopamine 会认为它们属于不同专辑。

解决：
  • 在 Dopamine 中右键歌曲 → "编辑标签"
  • 将"专辑艺术家"统一改为"Various Artists"或留空
  • 确保"专辑名称"完全一致

Q2：Dopamine 启动速度慢怎么办？
优化建议：
  • 减少音乐库文件夹数量，尽量合并到一个根目录
  • 在设置中关闭"启动时检查文件变化"（改为手动刷新）
  • 确保电脑有足够的内存（关闭其他占用内存的程序）

Q3：如何显示歌词？
  • Dopamine 3.0 版本已支持歌词功能：
  • 确保歌曲文件内嵌了歌词标签，或同目录下有 .lrc 文件
  • 在播放界面点击歌词按钮（如果有）
  • 或开启在线歌词自动下载（设置 → 歌词 → 启用在线歌词）

Q4：杀毒软件报毒怎么办？
由于 Dopamine 是开源软件，没有进行商业数字签名，部分杀毒软件可能会误报。

解决：
  • 将 Dopamine 安装目录添加到杀毒软件白名单
  • 从 GitHub 官方页面下载可确保文件安全
  • 如不放心，可查阅 GitHub 上的源代码自行编译
