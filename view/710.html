首次使用配置：手机端设置详解

软件安装好后，还不能立即使用，需要先在安卓手机端开启开发者模式。这是最关键的一步，很多新手卡在这里，其实非常简单！

1) 开启手机开发者选项（通用步骤）

不同品牌手机路径略有差异，但大体相同 ：
步骤 1：开启开发者模式
  • 打开手机"设置"
  • 找到"关于手机"（或"我的设备"）
  • 找到"版本号"（或"软件版本"、"MIUI 版本"等）
  • 连续快速点击 7 次（会提示"您已处于开发者模式"）
步骤 2：打开 USB 调试
  • 返回设置主界面
  • 进入"系统" → "开发者选项"（或直接在设置中搜索"开发者"）
  • 找到"USB 调试"开关，将其打开
  • 弹出警告时点击"确定"
步骤 3：允许 USB 调试授权（首次连接）
  • 用数据线将手机连接到电脑
  • 手机会弹出"允许 USB 调试吗？"的提示框
  • 勾选"始终允许使用这台计算机进行调试"
  • 点击"确定"

2) 连接使用

手机设置完成后，就可以开始投屏了！USB 有线连接是 Escrcpy 最稳定的连接方式，推荐新手：
  • 连接设备：用 USB 数据线将手机连接到电脑（建议使用原装线或质量较好的数据线）
  • 启动软件：打开 Escrcpy，等待几秒钟
  • 识别设备：在软件主界面的设备列表中，你应该能看到你的手机型号
  • 开始镜像：点击设备旁边的"开始镜像"按钮
  • 成功投屏：此时电脑会弹出一个新窗口，显示你的手机屏幕，直接用鼠标操作即可！

Escrcpy 还支持无线连接，让你摆脱线缆束缚，适合办公演示等场景。设置稍微复杂一点，但一劳永逸。

前提条件：
  • 手机和电脑连接到同一个 Wi-Fi 网络（同一局域网）
  • 安卓 11 及以上系统（部分旧系统需要额外设置）

连接步骤：
方法 A：扫码快速连接（推荐）
  • 先用 USB 线连接一次手机（完成首次授权）
  • 在手机"开发者选项"中进入"无线调试"
  • 点击"使用二维码配对设备"
  • 在 Escrcpy 软件中点击扫码连接，对准手机二维码扫描
  • 配对成功后，拔掉数据线，以后自动无线连接
方法 B：手动输入 IP 连接
  • 在手机"无线调试"页面查看 IP 地址和端口号（如 192.168.1.5:5555）
  • 在 Escrcpy 中输入该地址和端口
  • 点击"连接设备"，等待连接成功
  • 点击"开始镜像"即可无线投屏

常见问题与解决方案

Q1：软件提示"ADB 无法识别设备"怎么办？
  • 检查 USB 调试是否真正开启（有时需要重启手机）
  • 更换 USB 数据线（部分线只能充电不能传数据）
  • 重新插拔手机，确保授权弹窗点击了"允许"
  • Windows 用户可能需要手动安装手机驱动（如小米手机需安装 MiFlash 驱动）

Q2：无线连接总是失败？
  • 确保手机和电脑在同一 Wi-Fi 下（不能一个连 2.4G 一个连 5G，即使是同一个路由器）
  • 首次无线连接前，必须先通过 USB 连接一次完成配对
  • 检查手机"无线调试"页面是否显示已配对设备

Q3：画面卡顿、延迟高？
  • 降低分辨率和帧率设置
  • 关闭电脑的 GPU 硬件加速（在设置中）
  • 使用有线连接代替无线
  • 确保手机性能充足（老旧手机可能带不动高帧率）

Q4：macOS 提示"无法验证开发者"？
  • 前往"系统设置" → "隐私与安全性" → 找到"已阻止使用 Escrcpy" → 点击"仍要打开"
