安装Czkawka

步骤1：下载 windows_czkawka_gui_gtk.zip 的压缩包。

步骤2：解压文件。右键点击下载的 ZIP 文件，选择"解压到当前文件夹"（需要 WinRAR 或 7-Zip 等解压软件）
建议将解压后的文件夹移动到 C:\Program Files\Czkawka 或 D:\Tools\Czkawka 等固定位置，方便日后使用。


步骤3：双击 czkawka_gui.exe 即可启动。如果 Windows 提示"未知发布者"，点击"更多信息"→"仍要运行"即可。


步骤4：设置中文界面（首次启动）
  • 打开软件后，点击顶部菜单的 Settings（设置）
  • 找到 Language（语言）选项
  • 在下拉菜单中选择 简体中文 或 繁体中文
  • 重启软件即可生效

使用Czkawka的注意事项

1) 首次使用先备份：在清理重要文件夹（如工作文档、家庭照片）前，务必先备份到移动硬盘或云盘。

2) 先扫描，别急着删：第一次扫描后，不要立即删除，先检查一遍结果，确保没有误判。

3) 使用"移动到"而非"删除"：对于不确定的文件，先用 Move 功能移到临时文件夹，观察几天系统是否正常，再决定是否彻底删除。

4) 不要扫描系统目录：Windows 的 C:\Windows、C:\Program Files，macOS 的 /System，Linux 的 /bin、/sbin 等系统目录千万不要扫描，以免误删系统文件导致电脑无法启动。

5) 关闭其他程序：扫描时关闭正在使用的软件，特别是微信、QQ、钉钉等，因为它们可能正在写入文件，导致扫描结果不准确。

Czkawka常见问题

问题1：程序无法启动，提示缺少 DLL 文件
解决方法：下载 Visual C++ Redistributable 运行库并安装，或换用 MSYS2 版本。

问题2：界面显示模糊（高 DPI 屏幕）
解决方法：右键 czkawka_gui.exe → 属性 → 兼容性 → 更改高 DPI 设置 → 勾选"替代高 DPI 缩放行为"。

问题3：扫描速度很慢
解决方法：在设置中开启"使用缓存"功能，第二次扫描同目录会快 4-10 倍。
