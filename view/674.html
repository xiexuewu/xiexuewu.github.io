VS2026使用教程（使用VS2026编写C语言程序）

VS2026下载安装保姆级教程（附安装包+图文步骤） VS2022下载和安装教程（图解版）

继《VS2026下载地址和安装教程》之后，本节教大家如何用 VS2026 运行 C 语言程序。

例如，在 VS2026 中编写并运行下面的 C 语言代码：
#include <stdio.h>
int main()
{
    puts("Hello,World!");
    return 0;
}
需要经历如下几步操作。

1、创建项目

打开 VS2026，可以看到如下界面：


图 1 进入 VS2026 主界面

点击“继续但无代码”，跳转到 VS2026 的主界面：


图 2 新建项目

在菜单中依次选择“文件->新建->项目”：


图 3 创建项目
在弹出的如下窗口中选择“空项目”：

图 4 创建空项目

可以自定义项目的名称和存储位置：

图 5 自定义项目名称和存储位置

创建完成后，可以在【解决方案资源管理器】里看到整个项目，如下图所示：


图 6 创建好的项目

2、添加源文件

在 “源文件” 处右击鼠标，弹出的菜单中选择 “添加 --> 新建项” ，如下图所示：


图 7 新建源文件

在弹出的窗口里，自定义新建文件的名称（后缀名改为 .c），如下图所示：


图 8 自定义文件类型和名称

点击“添加”按钮，即可成功创建出一个名为 FileName.c 的源文件，如下图所示：


图 9 成功新建源文件

3、运行程序

将本文开头处的 C 语言程序添加到图 9 所示的 FileName.c 文件中，如下图所示：


图 10 编写 C 语言程序

点击“本地Windows调试器”，开始运行 FileName.c 文件，如下图所示：


图 11 一键运行 C 语言程序

如果程序没有错误，会看到程序的运行结果：

 

图 12 程序运行结果

总结

使用 VS2026 运行 C 语言程序，要经历“新建项目 -> 新建源文件 -> 编写 C 语言程序”共 3 个过程。

跟随本文操作 VS2026，你已经越过了第一道障碍，学会了如何编写代码，如何将代码生成可执行程序，这是一个完整的体验。
