新手常见问题与解决方案

Q1：软件无法启动或闪退
A：检查系统是否满足最低要求（Windows 需 64 位系统）。尝试以管理员身份运行，或检查杀毒软件是否误拦截。确保下载的是官方正版安装包 。

Q2：代码提示不显示或延迟
A：进入"工具"→"设置"→"编辑器设置"，检查语言服务是否开启。对于大型项目，首次打开时索引建立可能需要几分钟，请耐心等待。如遇到特定版本（如 4.66）的提示缓慢问题，建议升级到最新版本 。

Q3：浏览器预览无法打开
A：确保电脑安装了 Chrome 或其他现代浏览器。在"运行配置"中检查浏览器路径设置是否正确。如遇到端口冲突，尝试更换预览端口 。

Q4：真机调试连接失败
A：确保手机和电脑连接同一 WiFi 网络。检查手机是否开启了开发者模式和 USB 调试（Android）。如使用云打包功能，需先完成 DCloud 开发者账号的实名认证 。

Q5：插件安装失败
A：HBuilderX 支持部分 VS Code 插件，但建议优先使用官方插件市场的插件。手动安装插件时，需将插件文件解压至软件目录下的 plugins 文件夹内，然后重启软件 。
