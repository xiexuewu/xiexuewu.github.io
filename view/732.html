市面上投屏工具众多，但真正能做到免费、开源、高性能、零延迟的却凤毛麟角。QtScrcpy 正是这样一款脱颖而出的神器。

QtScrcpy 是一款基于 Scrcpy 内核开发的跨平台安卓实时投屏与控制工具，它允许用户通过 USB 数据线或 WiFi 无线网络，将安卓设备的屏幕实时投射到 Windows、macOS 或 Linux 电脑上，并且可以直接使用电脑的键盘和鼠标反向控制手机操作 。

在安卓投屏领域，主要有三款开源工具备受关注：Scrcpy（原版）、QtScrcpy 和 escrcpy：
  • Scrcpy 是性能天花板，延迟最低，但完全依赖命令行操作，对新手极不友好；
  • escrcpy 基于 Electron 开发，界面最现代化，但功能深度不如 QtScrcpy，且资源占用较高；
  • QtScrcpy 则在保持接近原生性能的同时，提供了最全面的功能集和清晰的操作界面，是从新手到专业用户都能满足的全能选择。
与原版 Scrcpy 相比，QtScrcpy 最大的改进在于提供了完整的图形用户界面（GUI），让不懂命令行的普通用户也能轻松上手。

使用QtScrcpy实现投屏

这里以 Windows 平台为例，为大家演示使用 QtScrcpy 实现投屏的过程：
1) 下载 QtScrcpy-win-x64-v3.3.3.zip （我的电脑是 64 位），完成解压。

2) 在解压的文件夹里找到 QtScrcpy.exe，双击启动 QtScrcpy。


3) QtScrcpy 可以通过 USB 连接线实现手机投屏，也可以借助 WiFi 实现。接下来先设置一下手机，手机的操作系统版本确定大于 5.0，这样才能正常投屏。

在安卓设备的开发者选项中启用USB调试，不同手机进入开发者选项的方式不一样，但都在设置里（我的在系统管理）：


通过 USB 将安卓设备连接到电脑，将看到设备编号已更新。


4) 点击“启动服务”，安卓屏幕就投屏到了电脑上：


5) 官网给出的无线连接步骤如下，大家感兴趣的可以自行尝试：
  1. 在安卓设备的开发者选项中启用USB调试
  2. 通过 USB 将安卓设备连接到电脑。
  3. 点击“更新设备”，您将看到设备编号已更新。
  4. 点击获取设备 IP
  5. 点击开始 adbd
  6. 点击无线连接
  7. 再次点击“更新设备”，系统会找到另一台具有 IP 地址的设备。选择该设备。
  8. 点击启动服务

注意：启动 adbd 后，无需保持 Android 设备通过 USB 连接。

QtScrcpy使用注意事项

1) 连接稳定性

  • USB 连接比 WiFi 更稳定、延迟更低，建议游戏或精细操作时优先使用 USB
  • WiFi 连接时，确保网络信号良好，避免频繁切换 WiFi 或距离路由器过远
  • 部分手机在锁屏或休眠后会断开 WiFi 连接，需要在手机设置中保持 WiFi 常亮

2) 性能优化

  • 如果电脑配置较低，可以在设置中降低投屏分辨率或帧率，减少卡顿
  • 关闭手机不必要的后台应用，释放系统资源
  • Windows 用户建议开启显卡硬件加速（如果软件支持）

3) 安全性提示

  • 首次连接时务必确认"允许 USB 调试"对话框，避免在公共电脑随意授权
  • 使用完毕后，建议关闭手机的 USB 调试开关，防止恶意软件利用
  • 不要在已 Root 的设备上存储敏感信息，虽然 QtScrcpy 本身不需要 Root
