动态内存管理

最佳归并树 动态内存管理
上一节 下一节
动态内存管理主要包含两方面内容，用户申请内存空间时，系统如何分配；用户使用内存空间完成后，系统如何及时回收。

不断分配和回收内存的过程，会产生诸多内存碎片。借助数据结构，内存碎片化的问题能够得到有效的解决。

无论是哪种操作系统，它的内存管理机制都使用了大量的数据结构知识，进而提高内存管理的效率。
